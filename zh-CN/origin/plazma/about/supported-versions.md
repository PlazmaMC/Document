# ✅ 版本支持状态

{% hint style="info" %}

**Plazma从1.19.4版本开始开发，1.19.4版本以下的版本不再开发。**

如果您正在寻找1.19.4以下的平台，请使用[Fusion (`1.19.2 ~ 1.19.3`)](https://github.com/RuinedTechnologyUnify/Fusion)或[Prismarine (`1.18.1 ~ 1.19.2`)](https://github.com/PrismarineTeam/Prismarine)。

{% endhint %}
