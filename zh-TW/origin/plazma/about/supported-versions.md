# ✅ 版本支持状态

{% hint style="info" %}

**Plazma 從 1.19.4 開始開發，1.19.4 以下的版本已不再開發。**

如果您正在尋找 1.19.4 以下的平台，請使用 [Fusion (`1.19.2 ~ 1.19.3`)](https://github.com/RuinedTechnologyUnify/Fusion) 或 [Prismarine (`1.18.1 ~ 1.19.2`)](https://github.com/PrismarineTeam/Prismarine)。

{% endhint %}
